How to Create Music with Sound Graphs

Select Points Click on any intersection in the grid to add a musical point. Each point plays both X and Y axis notes.
Create Sequences Connect multiple points to create flowing musical sequences. Notes play sequentially: A1, B1, A2, B2, ...
Fine-tune Rhythm Use selection mode to adjust individual note durations and create complex rhythmic patterns.
Experiment Freely Try different instruments and tempos to discover unique musical combinations.
